# Glossary

## A

**Ansible**: An open-source automation tool for configuration management and application deployment.

## I

**Infrastructure as Code (IaC)**: Managing infrastructure through code rather than manual processes.

## K

**Kubernetes (K8s)**: An open-source container orchestration platform.

## M

**MCP (Model Context Protocol)**: A protocol for connecting AI models to external tools and data sources.

**MicroK8s**: A lightweight, pure-upstream Kubernetes distribution.

## T

**Terraform**: An infrastructure as code tool for building, changing, and versioning infrastructure.

## Z

**Zammad**: An open-source helpdesk and ticketing system.
