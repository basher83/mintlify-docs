# API Reference

## Available Tools

### get_ticket

Retrieve a ticket by ID.

```json
{
  "tool": "get_ticket",
  "arguments": {
    "ticket_id": 123
  }
}
```

### create_ticket

Create a new ticket.

```json
{
  "tool": "create_ticket",
  "arguments": {
    "title": "Ticket title",
    "group": "Users",
    "customer": "customer@example.com"
  }
}
```

### search_tickets

Search for tickets.

```json
{
  "tool": "search_tickets",
  "arguments": {
    "query": "status:open"
  }
}
```
